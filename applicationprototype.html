Application Prototype

Overview

The ManaForge application prototype demonstrates how users can quickly and intuitively build Magic: The Gathering decks. It showcases the core features of the final product, allowing users to experiment with deckbuilding tools, search for cards, and preview how their deck performs with real-time synergy analysis.

Core Features in the Prototype

  • Deck Builder: Drag-and-drop interface for adding or removing cards.
  • Collection Tracker: Track owned cards and receive suggestions based on your collection.
  • Synergy Preview: See how cards interact and receive automatic suggestions for improving your deck.
  • Mana Curve Visualization: Displays mana curve and color distribution to ensure balanced gameplay.

Purpose of the Prototype

The prototype allows users to explore and test key features before full development. It helps identify usability issues, gather feedback, and ensure the final product meets player needs.

User Flow Demonstration

The prototype simulates the process of building a deck from start to finish: selecting a format, choosing cards, reviewing synergy suggestions, and saving the deck. This provides stakeholders and playtesters a clear picture of the application's intended functionality.
